# STM32-24LC16B-library

# Example:
Example folder contains (STM32CubeIde)project for STM32F4DISCOVERY. 

Connect EEPROM:

        A0,1,2 (PIN 1,2,3) -> not connected
               Vss (PIN 4) -> GND
               SDA (PIN 5) -> PB7 (pull-up to 3V)
               SCL (PIN 6) -> PB6 (pull-up to 3V)
               WP  (PIN 7) -> GND
               Vcc (PIN 8) -> 3V
